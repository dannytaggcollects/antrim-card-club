# Antrim Card Club website

This is a lightweight static website with free, browser-based content editing through Pages CMS.

## Client editing

1. Visit https://app.pagescms.org
2. Sign in with the GitHub account that owns this repository.
3. Open this repository and choose **Website text**.
4. Edit the labelled fields and press **Save**.
5. GitHub Pages will publish the saved content automatically.

The client should edit through Pages CMS only. They do not need to edit HTML, CSS, JavaScript, or JSON files.

### Adding links to text

Longer text fields use a rich-text editor. Select text and use the link button, or enter a Markdown-style link such as `[Visit Eventbrite](https://eventbrite.com/)`.

Navigation labels and destinations are editable under **Website text → General details → Navigation links**. For pages within this site, use filenames such as `about.html`. For external links, paste the full `https://` URL.

## First-time publishing

1. Create the repository in the client's GitHub account.
2. Upload or push the contents of this folder to the repository root.
3. In the repository, open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder, then save.
6. Sign in to Pages CMS and install its GitHub App for this repository.

## Eventbrite

In Pages CMS, open **Website text → Tickets page** and select one of:

- **Coming soon** — no booking link is shown.
- **Link directly to Eventbrite** — add the public Eventbrite URL.
- **Embed Eventbrite booking** — add the Eventbrite embed URL and public event URL.

## Contact details

Add the client's email address and social links under **Website text → Contact page** before launch.
